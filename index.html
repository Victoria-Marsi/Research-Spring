CSS Layouts

Flexbox Layout

Why flexbox?

Provides a more efficient way to layout, align, and distribute space among items in a container

Parent Properties Used

  • display > flex
  • flex-direction > establishes the main-axis (defining the direction flex items are placed in the flex container) > row
  • justify-content > defines the alignment along the main-axis > space-evenly
  • align-items > defines default behavior for how items are laid out along the cross-axis > baseline

Child Properties Used

  • align-self > allows the default alignment to be overriden for individual flex items > flex-start
